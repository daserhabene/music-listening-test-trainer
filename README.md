# 🎵 Music Listening Test Trainer - 듣기 시험 트레이너

서양음악사 시험 공부하기 귀찮아서 만든
**듣기 시험 연습용 웹앱**

---

## 이게 뭐냐면

* 곡을 듣고
* 작곡가 + 곡 제목을 맞히는
* **듣기시험 대비용 퀴즈 앱**

입니다.

---

## 핵심 기능

### 🎧 문제 출제

* 랜덤 (비복원)으로 곡 1개 재생
* 한 라운드 동안 모든 곡을 1번씩 출제

---

### ✏️ 정답 입력

* 작곡가 + 곡 제목 입력
* 오타 허용 (유사도 기반)
* 별칭(alias) 허용

예:

```javascript
title: "피아노 협주곡 5번 황제 1악장",
aliases: ["황제", "황제 1악장", "피아노 협주곡 5번"]
```

→ `"황제"`만 입력해도 정답 처리

---

### 🌀 연습 모드

* 라운드 단위 학습
* 진행도 표시 (예: 7 / 20)
* 한 바퀴 끝나면 자동으로 다음 회독 시작

표시 예:

```
2회독 · 누적 43문제
```

---

### 🧠 통계 시스템

두 가지 통계 제공:

* 🌀 이번 회독 통계
* 📊 전체 누적 통계

버튼으로 숨길 수 있음 (치트 방지용)

---

### ❌ 오답 시스템

* 전체 오답 리스트
* 이번 회독 오답 리스트

버튼:

* 전체 오답 다시
* 이번 회독 오답 다시

---

### 📝 시험 모드

* 모든 곡 1회씩 출제
* 정답 공개 없음
* 종료 시 결과 표시

예:

```
시험 종료! 점수: 17/20 (85%)

틀린 곡:
- 돈 조반니 서곡
- 클라리넷 협주곡 2악장
```

---

### 🎯 진행도 표시

* 진행바 UI
* 현재 위치 시각화

---

## playlist.js 구조

```javascript
playlist = [
  {
    composer: "베토벤",
    title: "피아노 협주곡 5번 황제 1악장",
    aliases: ["황제", "황제 1악장"],
    file: "audio/beethoven_emperor.mp3",
    start: 0,
    end: 60
  }
];
```

---

## 필드 설명

| 필드       | 설명     |
| -------- | ------ |
| composer | 작곡가    |
| title    | 대표 정답  |
| aliases  | 허용할 별칭 |
| file     | 오디오 경로 |
| start    | 시작 시간  |
| end      | 종료 시간  |

---

## 새로운 playlist 만들기
